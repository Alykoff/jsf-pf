Ключевые технологии/библиотеки задейственные в проекте jsf-pf:

  • JSF
  • CDI
  • JPA
  • LDAP
  • spring security 3
  • primefaces 5
  • maven/gradle
  • H2

Краткое описание

Проект предсталяет из себя шаблон веб приложения на JSF, для которого настроен spring security 3 и добавлена поддержка LDAP. При этом приложение может быть запущено на tomcat 7,8
